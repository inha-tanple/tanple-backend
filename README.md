## pre-requisite

1. install jdk-17
```bash
# install jdk-17
```

2. clone repository
```bash
git clone https://github.com/inha-tanple/tanple-backend.git
cd tanple-backend
```

## run

1. run h2-database

```bash
cd h2/bin

# Windows
./h2.bat

# macOS
./h2.sh
```

> you can access database in web <img src="docs/images/h2-setup.png" alt="h2-setup" style="width: 40%; display: block; margin: auto;">

2. run server
